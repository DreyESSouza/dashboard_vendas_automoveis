# Dashboard de Vendas Automotivas no Power BI

Dashboard interativo desenvolvido no Power BI com foco na análise de vendas automotivas, utilizando uma base de dados do Kaggle.

## Sobre o Projeto

Este foi meu primeiro dashboard desenvolvido de forma mais independente no Power BI.

Antes deste projeto, eu já havia desenvolvido um dashboard acompanhando aulas, mas este foi o primeiro em que consegui estruturar melhor a lógica, o layout e a construção da análise com mais autonomia.

Para este projeto, utilizei uma base de dados do Kaggle sobre vendas automotivas, realizando etapas de tratamento, tradução e padronização dos dados para facilitar a análise e melhorar a leitura das informações no dashboard.

O layout foi inspirado em uma referência apresentada durante meus estudos, mas com adaptações e alterações feitas por mim ao longo do desenvolvimento.

## Objetivo

O objetivo deste dashboard é permitir uma análise visual e prática dos dados de vendas automotivas, facilitando a identificação de padrões, desempenho de categorias, produtos, países e indicadores gerais de faturamento.

## Principais Recursos do Dashboard

- Visão geral de faturamento
- Indicadores principais (KPIs)
- Análise por categoria
- Análise por país
- Comparação de desempenho entre produtos
- Filtros interativos
- Visualização gráfica para apoio à tomada de decisão

## Etapas Desenvolvidas

Durante o desenvolvimento deste projeto, foram trabalhadas as seguintes etapas:

- Importação da base de dados
- Limpeza e tratamento dos dados
- Tradução dos campos
- Padronização das informações
- Modelagem para análise
- Construção dos visuais
- Organização do layout do dashboard

## Ferramentas Utilizadas

- Power BI
- Power Query
- DAX
- Kaggle
